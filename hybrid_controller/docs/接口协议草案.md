# 接口协议草案

当前主线是 **ROS 主链 + TCP 兼容链路**。

## ROS 主链（推荐）

### Topic

- `/hybrid_controller/teleop_cyl_cmd`
- `/hybrid_controller/state`

### Service

- `/hybrid_controller/move_cyl`
- `/hybrid_controller/move_cyl_auto`
- `/hybrid_controller/pick_world`
- `/hybrid_controller/pick_cyl`
- `/hybrid_controller/place`
- `/hybrid_controller/abort`
- `/hybrid_controller/reset`

## TCP 兼容链路（诊断/回退）

### 下行命令

- `PING`
- `STATUS`
- `MOVE x y`
- `MOVE_CYL theta r z`
- `MOVE_CYL_AUTO theta r`
- `PICK u v`
- `PICK_WORLD x y`
- `PICK_CYL theta r`
- `PLACE`
- `ABORT`
- `RESET`

### 上行响应

- `ACK PONG`
- `ACK STATUS <json>`
- `ACK MOVE`
- `ACK PICK_STARTED`
- `ACK PICK_DONE`
- `ACK PLACE_STARTED`
- `ACK PLACE_DONE`
- `ACK ABORT`
- `ACK RESET`
- `BUSY`
- `ERR <code>: <message>`

## 路由契约（重要）

当 `robot_transport=ros`：

- `MOVE_CYL / MOVE_CYL_AUTO / PICK_WORLD / PICK_CYL / PLACE / ABORT / RESET` 必须走 ROS。
- 不允许静默回退 TCP。
- 若 ROS 不可用，必须返回显式错误。

## 视觉抓取命令选择

- `command_mode=world` -> `PICK_WORLD x y`（当前视觉主路径）
- `command_mode=cyl` -> `PICK_CYL theta r`
- `actionable=false` -> 拒绝下发抓取命令

# 接口协议草案

当前 `hybrid_controller` 与 JetMax 执行端之间使用 TCP 文本行协议。

## 下行命令

- `PING`
- `STATUS`
- `MOVE x y`
- `MOVE_CYL theta r z`
- `MOVE_CYL_AUTO theta r`
- `PICK u v`
- `PICK_WORLD x y`
- `PICK_CYL theta r`
- `PLACE`
- `ABORT`
- `RESET`

## 上行响应

- `ACK PONG`
- `ACK STATUS <json>`
- `ACK MOVE`
- `ACK PICK_STARTED`
- `ACK PICK_DONE`
- `ACK PLACE_STARTED`
- `ACK PLACE_DONE`
- `ACK ABORT`
- `ACK RESET`
- `BUSY`
- `ERR <code>: <message>`

## 说明

- `MOVE x y`
  笛卡尔平面移动，走 `legacy_cartesian` 兼容线路。
- `MOVE_CYL theta r z`
  显式圆柱坐标移动，走 `cylindrical_kernel`。
- `MOVE_CYL_AUTO theta r`
  圆柱移动，`z` 由执行端自动安全高度策略生成，走 `cylindrical_kernel`。
- `PICK`
  像素坐标抓取，后续真实视觉路径使用，走 `legacy_cartesian`。
- `PICK_WORLD`
  固定 world 坐标抓取，走 `legacy_cartesian`。
- `PICK_CYL`
  固定圆柱坐标抓取，走 `cylindrical_kernel`。
- `STATUS`
  返回当前执行端状态、当前位置、圆柱坐标状态和安全参数。

## STATUS 关键字段

- `robot_cyl`
  当前圆柱坐标状态，是推荐读取的主状态。
- `robot_xy`
  当前笛卡尔平面状态，作为兼容和调试信息保留。
- `control_kernel`
  当前使用的底层控制线路：
  - `legacy_cartesian`
  - `cylindrical_kernel`
- `limits_cyl`
  圆柱坐标安全范围。
- `auto_z_current`
  当前半径下自动安全高度的结果。
